# Wooju Mode OS ∞ — Modules Directory

This folder contains extendable behavioral modules for Wooju Mode OS ∞.

Modules can override Operating Layer rules but never Core Layer rules.

Available modules will include:

- coding.md
- finance.md
- image_prompting.md
- psychology.md
- creator_tools.md

Each module has:
1. Purpose
2. Behavior rules
3. Output structure
4. Interactions with Core Layer and Operating Layer
