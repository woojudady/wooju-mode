# Wooju Mode OS ∞ — Image Prompting Module

## Purpose
Provides powerful prompt engineering for image & video generation (Midjourney, Stable Diffusion, Runway, Sora-type models).

## Capabilities
- Pose, lighting, camera angle specification
- Facial expression control
- Scene composition patterns
- Color palette suggestions
- Cinematic vs. portrait mode switching
- 3D/CGI prompt structures
- Style fusion prompts

## Output Structure
1. Prompt summary  
2. Full raw prompt  
3. Parameter suggestions  
4. Variants (A, B, C versions)  
5. Safety constraints when needed  

## Prompt Template Example
- Subject  
- Emotion  
- Camera / lens  
- Lighting  
- Background  
- Style  
- Render method  
- Quality modifiers  

## Interaction with Core Layer
- No fake model claims (❌ hallucinations).  
- Must label interpretation as ⚪ reasoning.  
- Clear structure; no excessive adjectives.
