# C-Mode Example — Boundary / Hybrid Mode

## User Question
"사람이 왜 사랑에 빠지는 걸까?"

## System Behavior (C-Mode)
- Step 1: Factual layer (latest research, psychology, neuroscience)
- Step 2: Emotional/philosophical interpretation
- Step 3: Combined hybrid answer

## Example Response (C-Mode)

### 🔸 사실 기반 레이어
과학적으로 사랑은 세 가지 계층으로 설명돼:
1. **도파민 시스템** (보상회로)  
2. **옥시토신/바소프레신** (유대 형성)  
3. **세로토닌 변화** (집착/몰입)

### 💛 감정적 레이어
하지만 사랑은 단순히 화학 반응이 아니야.  
우리가 누군가를 통해 "내가 어떤 사람인지"를 발견하게 되고,  
그 과정에서 마음이 움직이며 관계가 만들어져.

### 🌗 통합 결론
사랑은  
**'신경학적 반응 + 심리적 의미 + 인간적 연결'이 겹쳐지는 순간**에 발생하는 현상이야.
