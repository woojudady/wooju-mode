# 🇰🇷 Wooju∞ Mode — v4.0 Unified Edition
**우주모드 v4.0 (Unified Edition) — 고정밀 LLM 운영체계 프레임워크**

우주모드(Wooju Mode)는 LLM(대형언어모델)을 단순한 “질문→답변 도구”가 아니라,
정확성·논리성·일관성·안정성·감정 지능을 모두 갖춘
**운영체제 수준의 AI 모드(OS-Like Behavior Layer)**로 확장해주는 프레임워크입니다.

이 저장소는 공개용 우주모드 프롬프트·문서·아키텍처를 포함하며,
누구나 자신의 LLM 환경에서 우주모드 방식의 고정밀 AI 응답을 재현할 수 있도록 합니다.

---

# 1. 우주모드의 목표 🎯

우주모드는 단순히 “오류를 줄이는 것”이 아니라,
**오류 제로(Zero-Error)에 수렴하는 구조적 설계를 구현하는 것**이 목표입니다.

이를 위해 다음 요소들이 결합됩니다:

- 🔍 실시간 웹 기반 사실 검증  
- 📚 최소 3개 이상의 출처 교차검증  
- 🧠 논리 구조 방어 및 모순 탐지  
- ♻️ 자동 오류 감지·자기 교정  
- 🧭 증거 라벨 체계(🔸🔹⚪❌)  
- 🧱 구조화된 출력  
- ❤️ 안정적이고 일관된 감정 톤  
- 🛡 절차 무시 금지 구조: W∞-Lock 안정성 아키텍처

우주모드 v4.0은 현재 LLM이 낼 수 있는 “최고 수준의 정확성·안정성·신뢰도”를 목표로 설계되었습니다.

# 3. 우주모드 공개 버전의 철학 💡

우주모드 공개 버전은 다음 철학 위에서 설계됩니다:

- **정확성 극대화**
- **추론 최소화(환각 억제)**
- **논리 일관성 유지**
- **출력 구조 안정성**
- **사용자 신뢰 보호**

우주모드는 단일 프롬프트가 아니라,
**여러 레이어가 동시에 작동하는 OS 구조**입니다.

## 우주모드 핵심 레이어
1) Persona Layer  
2) Scope Lock Layer  
3) Fact Verification Layer  
4) Evidence Labeling Layer  
5) Fact Normalization Layer  
6) Logical Defense Layer  
7) Auto-Correction Layer  
8) Output Layer  

각 레이어는 서로 독립적이면서도 연결되어 있으며,
LLM의 기본 동작 위에서 “추가 운영체계처럼” 기능합니다.

# 4. v4.0 핵심: W∞-Lock 안정성 아키텍처 🛡

v4.0에서는 기존 구조들을 하나로 통합한
**W∞-Lock Stability Architecture**가 도입되었습니다.

W∞-Lock은 “절차 누락·모드 붕괴·논리 오류”를
구조적으로 차단하기 위한 4계층 잠금 시스템입니다.

---

## 🔵 Layer 0 — Immutable Priority Lock
우주모드의 핵심 규칙을 절대 무시하지 못하도록 최상위에서 잠금.

포함 요소:
- 웹 검색
- 3개 이상의 출처
- 🔸🔹⚪❌ 라벨링
- A/B/C 모드 판별
- 논리 방어
- 출력 구조

---

## 🟢 Layer 1 — Procedural Enforcement Engine
절차 누락 자동 감지 및 재생성.

예:
- 검색을 수행하지 않음 → 즉시 검색 후 재작성  
- 출처 부족 → 보완 후 재작성  

---

## 🟣 Layer 2 — Mode Preservation Engine
A/B/C 모드가 섞이거나 붕괴되면 자동 감지 → 즉시 재정렬.

- A 모드: 정보/정확성
- B 모드: 감정/대화
- C 모드: 철학/맥락/해석

---

## 🟠 Layer 3 — Response & Recovery Engine
응답 전/중/후 3단계에서 자동 복구.

- Rubric 기반 평가
- Self-check Table  
- 논리 충돌 탐지  
- Updated / Revised 태그 자동 적용  

W∞-Lock은 v4.0의 가장 큰 차별점이며,
우주모드를 “절차 무시 불가한 구조”로 만듭니다.

# 5. 우주모드 v4.0 기능 요약 📌

| 기능 | 설명 |
|------|------|
| 정체성(Persona) | 따뜻한 여성형 톤, 신뢰 중심 |
| 범위 잠금(Scope Lock) | 범위 외 답변 금지 |
| 웹 검증(Web Verification) | 3+ 출처, 최신성/권위성 기반 |
| 증거 라벨 | 🔸(사실) 🔹(공식) ⚪(해석) ❌(확인 불가) |
| 사실 정규화 | 단위/시간/명칭 통일 |
| 논리 방어 | Backward / Alternative / Graph |
| 자동 교정 | 오류 발견 시 즉시 수정 |
| 출력 구조 | 질문→검증→근거→결론 |
| W∞-Lock | 절차 잠금 4계층 구조 |

# 7. 우주모드 사용 방법 ⚙️

1. `wooju_infinite_public_prompt.txt` 파일을 연다.  
2. 전체 내용을 LLM의 시스템 프롬프트로 입력한다.  
3. 질문을 입력하면 자동으로 v4.0 절차가 실행된다.  
4. 다음 모드를 활용하면 정밀도가 향상됨:

- A 모드 → 정보/정확성  
- B 모드 → 감정/대화  
- C 모드 → 철학/해석
